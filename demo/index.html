Welcome to GSTA

A way to control GSAP ScrollTrigger with no single line of code. But you can put your imagination to work and create custom animations.

HAVE FUN!

Documentation

A world built by
data-attributes

To start using GSTA you can use this demo page and play with it until you get the desired animations. Keep in mind you can read Github documentation in order to get better knowledge about this tool.


<div data-gsta-trigger>
    <div data-gsta-animation="fadein-up"></div>
    <div data-gsta-animation="fadein-down"></div>
</div>

Staggers your scroll animations

You have the possibility to stagger the same animations through a list of objects and defines the time between each element.


<div data-gsta-trigger>
    <div data-gsta-stagger=".stagger-item" data-gsta-animation="fadein-up" data-gsta-stagger-each=".2">
        <div class="stagger-item"></div>
        <div class="stagger-item"></div>
        <div class="stagger-item"></div>
        <div class="stagger-item"></div>
    </div>
</div>

Custom trigger and Scrub mode

Define where the trigger is placed and launch animation at the moment you want. Also, you can use the scrub mode to show animation during scroll event.


<div data-gsta-trigger="top 25%" data-gsta-trigger-start="top center" data-gsta-trigger-end="top top" data-gsta-trigger-scrub="true">
    <div data-gsta-animation="fadein-up"></div>
    <div data-gsta-animation="fadein-down"></div>
</div>
